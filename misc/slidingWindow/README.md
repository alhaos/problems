# Sliding Window — Максимальная сумма подмассива

Поиск максимальной суммы подмассива фиксированной длины `k`.

## Интерфейс

```
func maxSum(nums []int, k int) int
```

## Пример

```
nums = [2, 1, 5, 1, 3, 2]
k    = 3

Ответ: 9   // [5, 1, 3]
```

```
nums = [2, 3, 4, 1, 5]
k    = 2

Ответ: 7   // [3, 4]
```

## Ограничения

- `1 <= k <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`
- Алгоритм должен работать за **O(n)**
