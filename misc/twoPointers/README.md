# Two Pointers — Два указателя

## Условие

Дан **отсортированный** массив целых чисел `numbers` (индексация с 1) и целевое число `target`.

Найди два числа, сумма которых равна `target`, и верни их индексы `[index1, index2]`, где `index1 < index2`.

## Пример

```
numbers = [2, 7, 11, 15]
target  = 9

Ответ: [1, 2]   // numbers[1] + numbers[2] = 2 + 7 = 9
```

## Ограничения

- `2 <= numbers.length <= 30 000`
- `-1000 <= numbers[i] <= 1000`
- Массив отсортирован **по возрастанию**
- Гарантируется ровно **одно решение**
- Нельзя использовать один и тот же элемент дважды
- Индексация с **1**

## Сигнатура функции

```go
func twoSum(numbers []int, target int) []int
```
