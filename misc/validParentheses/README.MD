# Valid Parentheses — Правильные скобки

Проверка корректности строки из скобок с помощью стека.

## Интерфейс

```
func isValid(s string) bool
```

## Пример

```
s = "()"       → true
s = "()[]{}"   → true
s = "(]"       → false
s = "([)]"     → false
s = "{[]}"     → true
```

## Ограничения

- `1 <= s.length <= 10^4`
- `s` содержит только символы `()[]{}`
