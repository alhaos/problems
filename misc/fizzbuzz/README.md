# FizzBuzzSum

## Условие

Напиши функцию `FizzBuzzSum(n int) int`, которая возвращает сумму всех чисел от
`1` до `n` включительно, которые **не** кратны ни 3, ни 5.

## Примеры

| n  | Числа                     | Результат |
|----|---------------------------|-----------|
| 1  | 1                         | 1         |
| 5  | 1, 2, 4                   | 7         |
| 10 | 1, 2, 4, 7, 8             | 22        |
| 15 | 1, 2, 4, 7, 8, 11, 13, 14 | 60        |

## Сигнатура

```go
func FizzBuzzSum(n int) int
```

## Сложность

- Время: O(n)
- Память: O(1)
