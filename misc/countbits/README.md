# CountBits

## Задача

Напиши функцию `CountBits(n int) int`, которая возвращает количество единичных битов в бинарном представлении числа `n`.

## Примеры

| n   | binary     | result |
|-----|------------|--------|
| 0   | `00000000` | 0      |
| 1   | `00000001` | 1      |
| 7   | `00000111` | 3      |
| 13  | `00001101` | 3      |
| 255 | `11111111` | 8      |

## Ограничения

- `n >= 0`
- Нельзя использовать `math/bits` и другие готовые функции подсчёта битов
- Целевая сложность: **O(k)**, где k — количество единичных битов

## Подсказка

Операция `n & (n-1)` сбрасывает **младший установленный бит** числа `n`.

Например:
```
n     = 1101  (13)
n-1   = 1100  (12)
n&n-1 = 1100  (12)  ← бит сброшен
```

Это позволяет итерировать ровно k раз, а не по всем 64 битам.
