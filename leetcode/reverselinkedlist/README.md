# 206. Reverse Linked List

**Difficulty:** Easy  
**Topic:** Linked List

## Problem

Given the head of a singly linked list, return the list reversed.

## Examples

| Input | Output |
|-------|--------|
| [1,2,3,4,5] | [5,4,3,2,1] |
| [1,2] | [2,1] |
| [] | [] |

## Constraints

- 0 <= number of nodes <= 5000
- -5000 <= Node.val <= 5000

## Approach

Iterative — три указателя.
