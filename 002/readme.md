### Алгоритм Е Алгоритм Евклида

Даны два целых положительных числа `M` и `N`. Требуется найти их наибольший общий делитель, т. е. наибольшее целое положительное число, которое нацело делит оба числа. 

``` powershell

& {
  <#
    .SYNOPSIS
        Алгоритм Е {Алгоритм Евклида). Даны два целых положительных числа M
        и N. Требуется найти их наибольший общий делитель, т. е. наибольшее целое 
        положительное число, которое нацело делит оба числа т и п. 
    .PARAMETER string
        [int]$M, [Int]$n
 
    .OUTPUTS
        [int]
#>

    param(
      [int]$m,
      [int]$n
    )

    $r = $m % $n

    while ($r -ne 0){
        $m, $n = $n, $r
        $r = $m % $n
    }
    return $n
    
  } 2247 12
  
  ```
